Политика конфиденциальности

Приложение "Счастливые числа" не собирает, не хранит и не передает персональные данные пользователей.

Что мы НЕ делаем:

  • Не запрашиваем регистрацию
  • Не собираем email, телефон, имя
  • Не храним историю на серверах
  • Не передаем данные третьим лицам

Реклама:

Приложение использует Яндекс Mobile Ads SDK для показа рекламы. Может использоваться рекламный идентификатор (Advertising ID) для персонализации рекламы.
